# DOT Protocol ◉

**The Open Eye. We see, we share.**

DOT is the open protocol beneath the next internet — a 153-byte signed, chained, verifiable observation primitive. Identity, communication, money, and reputation, with no company in the middle that owns your data. Free forever. Apache-2.0 / MIT / public domain across the stack.

## Start here

```bash
npm install @dotprotocol/sdk      # signing, compression, wrapping, identity
pip install dot-python            # the 153-byte format in Python
```

## npm

All packages live under the [`@dotprotocol`](https://www.npmjs.com/org/dotprotocol) scope. `@dotprotocol/sdk` is the one-install entry point.
